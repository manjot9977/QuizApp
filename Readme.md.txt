Quiz App with Kotlin and Firebase
This is a simple quiz app developed in Kotlin that leverages Firebase Realtime Database for quiz data storage. The app allows users to take quizzes from different categories and provides real-time feedback on their performance.

Features
User authentication with Firebase Authentication.
Fetches quiz data from Firebase Realtime Database.
Displays quiz questions with multiple-choice options.
Calculates and displays the user's score upon completing a quiz.
Prerequisites
Before you begin, ensure you have the following installed:

Android Studio
Kotlin
Firebase Account (Realtime Database)
